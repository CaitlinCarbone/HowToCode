The Basics

Steps

  1. Make sure you make a branch first to protect the current GitHub content.

  2. git checkout -b branch

  3. git status

  4. git add [filename.extension]

  5. git commit -m "Add message"

  6. git push origin [branch name]

  7. You have now successfully added content to your branch!

Helping Commands

  • git status is your best friend! You can check which files Git is tracking by running git status
  • git reset allows you to revert git add commands

Any Alternatives?

There are so many different ways to use Git and one recommendation is using GitHub Desktop for a modern user interface
